Using ``3-rectangle``
-------------------

This is an example text file in reStructuredText format.  First import
``Rectangle`` from the ``3-rectangle`` module:

    >>> Rectangle = __import__('3-rectangle').Rectangle

Now use it:

    >>> my_rectangle = Rectangle(2, 4)
    >>> my_rectangle.__dict__ == {'_Rectangle__height': 4, '_Rectangle__width': 2}
    True

    >>> my_rectangle.width = 10
    >>> my_rectangle.height = 3
    >>> my_rectangle.__dict__ == {'_Rectangle__height': 3, '_Rectangle__width': 10}
    True

    >>> try:
    ...     print(Rectangle(4, "School"))
    ... except Exception as e:
    ...    	print(e)
    height must be an integer

    >>> try:
    ...     print(Rectangle("School", 4))
    ... except Exception as e:
    ...    	print(e)
    width must be an integer

    >>> try:
    ...     print(Rectangle(-1, 4))
    ... except Exception as e:
    ...    	print(e)
    width must be >= 0

    >>> try:
    ...     print(Rectangle(5, -4))
    ... except Exception as e:
    ...    	print(e)
    height must be >= 0
